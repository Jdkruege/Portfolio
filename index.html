Education

Below I have chodsen to highlight some of the more notable classes that I have taken at Iowa State. These classes are ones that I feel have influenced my education and have given me valuable experience outside of the generic programming track.

Intro to Computer Graphics

This course gave me the fundamentals of how render objects in 3D space. It taught me about shaders, matrix manipulations, applying textures, and how to apply these concepts in a browser. This class taught me how to program graphics using WebGL via javascript, and briefly mentioned the intricacies of higher level languages. For the final presentation in this class I prototyped an algorithm that would create smooth randomly generated terrain using sample terrain tiles.

Computer Game Programming

After learning about graphics I decided to pursue a class dedicated to teaching the concepts of programming games. This class gave me introductions to XNA, Unity, AI algorithms, and some minor modelling in blender. I took part in a semester long professional project where we marketed, and developed, a game for the Battleship Iowa museum, working with a staff member on the ship. I also researched and presented a 30 minute tutorial on game creation using CryEngine 3, showing students how to build a simple game.

Principles of Programming Languages

Principles of Programming Languages was a class that focused on breaking down the common themes present in modern scripting languages. We studied an inhouse scripting language built using Java. The common idea of this course was to introduce a concept of a language, look at how to implement it, and then work with and extend the language. During our coursework we explored concepts such as: Grammars, Interpreters, Memory, Garbage Collection, Lambdas, Type checking, and variables. This class gave me an exceptional look into the mechanics that form languages I use everyday.

Software Project Management

Software Project Management was a class that sought to teach students about the core responsibilites of a project manager. In this course students were split up into groups for four unique projects, in which each student would be given the chance to be a manager. It was through the coursework and the projects that I was able to get a better understanding of team management, and truly understand all the responsibilities of a manager. We learned processes of production, project plan creation, design documentation, review standards, testing, and how to be a good manager.

Software Testing

Software Testing taught me so much about the testing process. It was during this course that I learned not only how to write sound, complete test, but also how to write code that was testable. Coursework also taught me about various methods of software testing, as well as ways to write, evaluate, and manage software test.

Technical Communication

Professional communication is something that all professionals should know, and it is a skill that software engineers need to know. Coursework in this class aimed to teach us styles for writing internal and external emails, analyzing documents for correctness, and the skills needed to conduct presentations in focused mannner.

Linear Algebra

Linear algebra was a course that helped teach me the fundamentals I needed for computer graphics. It gave me the insight needed to do matrix multiplication, and taught me skills that helped manipulate objects in computer generated 3d space.

General Education Reflection

For general education I took: Economics of Discrimination, World Religions, and Ethics

In the short term, I hope to graduate college and get a job working at a game development company. I view the environment as extremely fun, and I think game developers provide enjoyment for many people. The work environment is very relaxed and the work sounds like it will be fun. I would love to be involved in a company like this, helping make games that people enjoy. The work requires knowledge of math and graphics, as well as programming challenges that occur with trying to add new ways to play.

Ethics is a course that I believe will impact my future. It was through this course that I learned about moral dilemmas, and the tools needed to evaluate said dilemmas. This course taught me that there are multiple ways to view situations, and multiple ways to approach situations to ensure the most morally upright path is taken.

I've had three software engineering courses that made a point to cover ethics, and my previous coursework helped me during these classes. Ethics is a very important concept in software engineering, as everyday more and more objects in our world are being run by computer programs. The wide spread effects of code that was built unethically could result in the deaths of a single hosptial patient who is undergoing treatments for a life threantening condition to the deaths of many thousands of people due to poorly managed nuclear control codes. The impacts of all decisions that we make in software design and implementation have far-ranging effects that we must consider, which is where a good handle on ethics comes in. I view my enrollment in an ethics course as a plan for my future creating ethically appropriate code.

An example of applied ethics in action, occured during a project management course at ISU. The course had the class split up into teams to create a system that allowed teachers to automatically take attendance without all the tedium normally needed. During our project plan we had to come up with ethical ways of obtaining photos of each student for use by our facial recognition software. I view this as an example of applied ethics while creating an engineering solution.
